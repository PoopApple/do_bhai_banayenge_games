# 🎮 PriSet.io — Retro Gaming Website

**PriSet.io** is a web-based retro arcade platform that revives the magic of 8-bit games using modern web technologies. Built with **HTML**, **CSS**, **JavaScript**, **GSAP**, **Framer Motion**, **Flask**, and **MongoDB**, it currently features playable versions of **Flappy Bird** and **Simon Says**, with more games coming soon!

---

## 🌟 Features

- 🎯 Retro-styled UI with pixel fonts and classic animations
- 🕹️ Fully playable **Flappy Bird** and **Simon Says** games
- 🌈 Smooth animations powered by **GSAP** and **Framer Motion**
- 🔗 Flask backend for routing, game logic, and score tracking
- 📦 MongoDB integration for storing user data and scores
- 💻 Responsive design, cross-browser support
